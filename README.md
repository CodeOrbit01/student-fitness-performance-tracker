# Student Fitness & Performance Tracker

A full-stack fitness tracking web application that enables students to monitor workouts, sleep, water intake, daily steps, and overall fitness progress through dynamic dashboards and analytics.

## Features

- Workout Logging.
- Sleep Tracking
- Step Tracking
- Water Intake Tracking
- Analytics Dashboard
- Progress Tracking
- Goal Completion Monitoring
- Responsive Bootstrap UI
- RESTful APIs

## Tech Stack

### Frontend

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Chart.js

### Backend

- Java 17
- Spring Boot
- Spring Data JPA
- Lombok

### Database

- MySQL
